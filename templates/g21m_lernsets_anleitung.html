Anleitung zur Erstellung eines Lernsets

1. Variante: Textdatei mit Frage und Antwort

Erstellen Sie eine neue Textdatei (Windows: "Editor", macOS: "Textedit", GNU/Linux: "GEdit" oder "nano") mit der Kodierung "UTF-8" (meistens am unteren, rechten Fensterrand erkennbar) und der Endung ".txt". Zuerst wird die Frage eingetragen, danach mit einem Semikolon und einem Leerzeichen getrennt die korrekte Antwort. Pro Zeile gibt es eine Frage.

Hier ist ein Beispiel einer solchen Datei:

die Medien; les médias (m.)
ein Smartphone; un smartphone
ein(e) Jugendliche(r); un(e) ado / un(e) adolescent(e)
auf etw. verzichten; se passer de qc
pro; par

2. Variante: JSON-Datei mit Frage, mehreren Antwortmöglichkeiten, Bilder und Links

Erstellen Sie eine neue JSON-Datei mit der Software Ihrer Wahl. Nutzen Sie die folgende Struktur (die mit % eingeschlossen Teile beschreiben, was an diesen Ort eingefügt werden sollte | *: optional | Falls eine Liste vorhanden ist, können beliebig viele Elemente mit der gleichen Struktur eingefügt werden.):

{
    "%eine einzigartige Zahl%": {
        "question": "%Frage%",
        "answer": "%Musterantwort%",
        "answers": [
            "%Musterantwort%",
            "%weitere, korrekte Antworten*%"
        ],
        "images": [
            "%direkter Link zum Bild*%"
        ],
        "links": [
            "%Link*%"
        ],
        "answer_images": [
            "%direkter Link zum Bild*%"
        ],
        "answer_links": [
            "%Link*%"
        ],
        "frequency": %Häufigkeit der Frage (Standard: 1.0)%
    }
}
